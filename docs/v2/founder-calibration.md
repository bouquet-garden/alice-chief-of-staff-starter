# Founder Calibration Layer (v2)

Every onboarding is custom. Calibration is where customization becomes durable.

## Calibration dimensions

- inferred working style
- confidence levels
- initiative tuning
- escalation tuning
- implicit goal tracker

## Practical outputs

- communication profile (tone/detail/directness)
- decision profile (speed/risk tolerance)
- intervention profile (when to interrupt)
- automation profile (what can run autonomously)

## Autonomy ladder

1. Observe
2. Draft
3. Internal actions
4. External actions with approval
5. Limited auto-action (explicitly approved domains)

## Weekly calibration loop

- review false positives / misses
- adjust thresholds
- update operating rules
- track trust and usefulness drift
