# Compound Learning + Nightshift (v2)

This module turns Alice from static setup into compounding operator.

## Compound learning loops

- incident -> rule
- experiment -> scorecard
- workflow success -> playbook
- recurring friction -> roadmap item

## Nightshift principles

- budget-capped
- reversible outputs first
- monitor for stalls
- morning digest with structured handoff

## Nightshift task mix

- communications prep
- strategic research
- coding/build tasks (where applicable)
- operations reconciliation
- decision prep artifacts

## Safety guardrails

- drafts over sends
- branch/PR over direct production change
- explicit escalation thresholds

## Moat effect

The system gets better every week through documented learning and operational feedback.
