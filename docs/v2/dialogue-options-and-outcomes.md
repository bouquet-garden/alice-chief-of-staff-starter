# Dialogue Options & Outcomes (Product Settings)

These settings are core product controls, not optional fluff.

## 1) Autonomy ladder

Options:
- Observe
- Draft
- Internal actions
- External actions with approval
- Bounded auto-action

Recommended default: **Draft + Internal actions**

Expected outcome:
- high trust during early adoption
- reduced risk of accidental external action

## 2) Proactivity level

Options:
- Low (P0/P1 only)
- Medium (daily summary + notable risks)
- High (proactive recommendations + staged drafts)

Recommended default: **Medium**

Expected outcome:
- useful signal without notification fatigue

## 3) Escalation policy

Options:
- immediate for legal/finance/public commitments
- digest for low-risk drift
- no escalation for purely formatting/tagging tasks

Recommended default: **Immediate for high-risk categories**

Expected outcome:
- safer operation with fewer unnecessary interruptions

## 4) Source-of-truth weighting

Options (example):
- Notion: execution authority
- Email: external commitment authority
- Slack: signal only
- Obsidian/markdown: strategy authority
- transcripts: decision evidence

Recommended default: this example set

Expected outcome:
- fewer contradictory recommendations
- faster conflict resolution

## 5) Communication style

Options:
- terse / strategic / detailed
- recommendation strength: suggest / recommend / strongly recommend
- pushback level: low / normal / high

Recommended default: **strategic + recommend + normal pushback**

Expected outcome:
- lower rewrite rate and better founder fit

## 6) Quiet hours and interruption windows

Options:
- strict quiet window with P0 bypass only
- normal delivery during work hours

Recommended default: **strict quiet hours, P0 bypass**

Expected outcome:
- less noise, better trust in proactive alerts
