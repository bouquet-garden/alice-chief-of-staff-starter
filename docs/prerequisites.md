# Prerequisites

## Minimum
- OpenClaw running
- one connected messaging surface
- git installed
- curl installed

## Strongly recommended
- browser access enabled in OpenClaw
- a writable workspace directory
- Notion, email, calendar, Slack, or Obsidian available as real source systems

## Optional: GBrain
Required:
- `bun`
- disk space for a local PGLite brain

Nice to have:
- `OPENAI_API_KEY` for embeddings

Without an OpenAI key, GBrain is still useful in keyword-search mode.

## Optional: Gstack
Required:
- Claude-oriented coding setup
- `~/.claude/skills/` available

Use Gstack only for medium/heavy coding workflows. It is not necessary for non-coding use cases.

## Product recommendation
Do not block onboarding on every optional integration being perfect.
A real first magic moment is worth more than a complete setup checklist.
