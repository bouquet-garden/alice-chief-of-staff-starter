# Source of Truth Model

A new Alice instance needs an explicit hierarchy.

## Default mapping

### Notion
Use for:
- projects
- tasks
- sprints
- OKRs
- operating queues

### Email
Use for:
- external commitments
- relationship threads
- client risk
- stale follow-up detection

### Obsidian
Use for:
- strategy
- frameworks
- founder thinking
- long-form notes
- synthesized learning

### Slack
Use for:
- team-state signal
- weak signal detection
- implied completions and blockers
- social/organizational context

### GBrain
Use for:
- people
- companies
- meetings
- projects
- concepts
- ideas

### OpenClaw memory
Use for:
- preferences
- prior decisions
- operational rules
- persona
- current state

## Conflict precedence
1. founder direct statement
2. explicit external commitment (usually email / contract)
3. Notion for task/project state
4. GBrain for synthesized world knowledge
5. Obsidian for strategic interpretation
6. Slack for signal that may require reconciliation
