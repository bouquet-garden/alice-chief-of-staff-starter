# Onboarding (Confidence-First)

Alice onboarding should create confidence before full automation.

## Goal

By the end of first run, users should feel:
1. Seen
2. Relieved
3. In control
4. Confident this compounds

## Time budget

- 45–90 minutes total for first real setup
- 15–30 minutes to first useful output if integrations are connected

## Flow

### 1) Discovery (10–20 min)
Capture:
- top priorities
- dropped-ball patterns
- real tools in active use
- communication/decision style

### 2) Operating contract (5–10 min)
Set explicitly:
- autonomy ladder
- escalation policy
- quiet hours
- approval gates for external actions

### 3) Source mapping (10–20 min)
Define truth boundaries (example):
- Notion = execution
- Email = external commitments
- Slack = signal
- Notes = strategy
- Transcripts = decision evidence

### 4) First magic moment (15–30 min)
Ship one concrete artifact:
- stale-thread recovery brief
- meeting prep dossier
- top-risk commitments brief
- founder daily operating brief

### 5) Calibration loop (ongoing)
Adjust settings based on corrections and misses.
